<h2 id="instalacao"> 💻 Rodando localmente </h2>

Clone o projeto

```bash
  git clone https://github.com/JorgeRodrigo481516/survey-form.git
```

Entre no diretório do projeto

```bash
  cd survey-form
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
